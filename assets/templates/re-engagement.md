# Re-engagement Template — VRASHOWS

**Uso:** Contatos que não responderam após ciclo completo (cold + 2 follow-ups) ou inativos há 30+ dias.
**Attachment:** Reenviar PDF apenas se houve resposta anterior sem conversão.
**Tom:** Renovar interesse com contexto novo — nunca parecer insistente.

---

## Subject line options

- `[Nome] — atualização VRASHOWS para [Ano]`
- `Nova estrutura operacional — [Evento Relevante]`
- `Algo mudou desde nosso último contato`

---

## Body (PT-BR)

### Variante 1 — Gatilho de evento

```
Olá [Nome],

Passaram alguns meses desde meu último contato.

Com [Evento/Feira] se aproximando, queria retomar — a demanda por operação especializada em feiras enterprise está crescendo e a janela para estruturar parcerias está se fechando.

Se fizer sentido revisitar, estou disponível para uma conversa rápida.
```

### Variante 2 — Nova capacidade

```
Olá [Nome],

Desde nosso último contato, expandimos nossa estrutura operacional para incluir [nova capability — ex: transporte executivo, lounge premium, staff bilíngue].

Achei relevante compartilhar dado o perfil de eventos da [Empresa].

Posso enviar uma atualização rápida do nosso portfólio?
```

---

## Rules

- Acknowledge gap in contact naturally ("alguns meses")
- Bring new context — don't repeat original pitch verbatim
- Maximum 100 words
- If no response to re-engagement, mark as `disqualified` and respect the unsubscribe
